## 🚀 Como Executar Localmente

### Pré-requisitos
- Um navegador moderno (Chrome, Firefox, Safari, Edge)
- Um editor de texto ou IDE (opcional)
- Python ou outro servidor HTTP local (opcional, para melhor experiência)

### Instalação

1. Clone o repositório:
```bash
git clone https://github.com/devbatista/devbatista_site.git
cd devbatista_site
```

2. Abra o arquivo `index.html` no navegador:
```bash
# macOS
open index.html

# Linux
xdg-open index.html

# Windows
start index.html
```

Ou use um servidor local:
```bash
# Python 3
python3 -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```

Depois acesse `http://localhost:8000` no seu navegador.
